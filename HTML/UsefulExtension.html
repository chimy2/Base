유용한 VS Code Extension

1. Material Theme

  • UI 화면의 색상을 자신의 눈에 편안한 테마로 설정

2. Material Icon Theme

  • 파일들이 어떤 파일인지 눈에 들어오도록 아이콘을 추가해준다

3. Prettier

  • 코드를 자동으로 저장된 형식대로 나열해준다

4. Bracket Pair Color

  • 괄호마다 다른 색을 부여해 구분이 쉽게 해준다

5. Indent-Rainbow

  • 들여쓰기의 구분이 쉽도록 도와준다

6. Auto Rename Tag

  • HTML의 opening tag나 closing tag 변경 시 변경된 tag에 맞게 앞뒤 태그를 변경해줌

7. CSS Peek

  • 해당 CSS 파일이 어디있는지 빠르게 찾아준다

8. HTML CSS Support

  • CSS에서 정의한 명칭들을 HTML파일에서 자동으로 찾아준다

9. HTML to CSS autocompletion

  • HTML에서 정의한 class명을 CSS파일에서 자동으로 찾아준다

10. Live Server

  • 새로고침할 필요 없이 파일을 수정하면 변경사항을 바로바로 보여준다

내장된 유용한 기능

1. Markdown Preview

  • markdown을 확인하면서 작성 가능
  • ctrl+shift+p > Markdown: Open Preview

2. Emmet

  • Zen Coding이라고 불렸음
  • HTML,xml, xsl 편집을 빠르게 할 수 있도록 도와주는 플러그인
  • 텍스트 에디터에 기본 내장 되어 있는 경우가 많다

1) 사용법

  • ! : 기본 html 코드
  • .className : 클래스명이 포함된 div 생성
  • #idName : 아이디명이 포함된 div 생성
  • tag.className : 클래스명이 포함된 Tag 생성
  • tag#idName : 아이디명이 포함된 Tag 생성
  • parentTag > childTag : 부모 태그(부모 노드)와 부모 태그 안에 자식 태그(자식 노드) 생성
  • siblingTag + siblingTag : 형제 태그(형제 노드)를 연이어 생성
  • tag * repeatNumber : 태그를 반복할 수만큼 생성
  • parentTag > childTag ^ ParentTag : 부모태그와 자식태그 생성 후 다시 상위 태그로 올라가서 부모 태그와 나란히 생성
  • (tag+tag) : 태그를 그룹화하여 사용가능
  • tag{tagcontent} : 태그 내용과 함께 생성
  • tag.className$, tag#idName$, tag{$} : 숫자 반복
  • lorem : 의미 없는 content 자동작성
  • lorem3 : 의미 없는 content의 3단어 자동작성
