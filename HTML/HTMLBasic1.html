HTML Basic

HTML

  • HyperText Markup Language
  • 웹 브라우저 상에서 보여지도록 설계된 문서의 표준 마크업 언어
  • Web Content의 의미와 구조 결정
  • CSS와 Javascript의 도움을 받을 수 있음

Markup language

  • 일반적인 text와 구분하기 위해서 문서에 annotating한 시스템
  • 문서의 논리적 구조와 배치 양식에 대한 정보를 표현하는 언어

jsbin
  • 개발툴을 설치하지 않아도 간단한 구현을 도와주는 웹 사이트
  • https://jsbin.com

Html 구조

<!DOCTYPE html> - Html문서임을 알려주는 문구
<html> - 가장 상위 태그로 head와 body로 나뉨
    <head> - 사용자에게 보여지는 UI적인 요소가 없고 Title, 부가설명, 제목, 아이콘들 정의 및 css 파일 연결 
             -> meta 데이터만 존재
    <meta charset="utf-8"> - 현 페이지에서 사용하는 글자의 포맷 설정
    <meta name="viewport" content="width=device-width"> - viewport 정의
    <title">제목</title"> - 페이지 제목 정의
    </head>
    <body> - 사용자에게 보여지는 가장 중요한 최상위 container
             -> body태그 안에 작성한 내용들이 사용자에게 보여진다
    </body>
</html>

W3C

  • World Wide Web Consortium
  • WWW의 페이지 기술 언어(PDL, Page Description Language)인 HTML 표준화 진행
  • 웹 브라우저와 서버 사이의 통신 규약인 하이퍼텍스트 전송 규약(HTTP) 표준화 진행
  • W3C의 웹 표준화를 통해 모든 웹 브라우저들이 표준에 맞게 공통 구현

MDN

  • 모질라에서 제공하는 웹기술 정보를 알려주는 웹사이트
  • 가장 빠르게 최신 웹 개발 기술을 확인할 수 있다
  • https://developer.mozilla.org

validator

  • 유효한 태그를 사용하는지 체크해주는 웹사이트
