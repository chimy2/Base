CSS의 구조

Selector {
    Property: Property value; <- Declaration
}

CSS 선택자(selector)

  1. Type selector
    • tag { }
    • 해당하는 모든 HTML 태그 선택
  2. Universal selector
    • * { }
    • 문서의 모든 항목 선택
  3. Class selector
    • .class { }
    • class 속성값과 같은 항목 선택
  4. id selector
    • #id { }
    • id 속성값과 같은 항목 선택
  5. Attribute selector
    • tag[attribute] { }
    • 해당 태그의 해당 속성값과 같은 항목 선택
  6. Pseudo-class selectors
    • tag:state { }
    • 해당 태그가 특정 상태인 항목 선택
  7. Pseudo-element selectors
    • tag::part { }
    • 해당 태그의 특정 부분 선택
  8. Descendant combinator
    • selector1 selector2 { }
    • 특정 부모의 자식 선택
  9. Child combinator
    • selector1 > selector2 { }
    • 특정 부모의 첫번째 하위요소 선택
  10. Adjacent sibling combinator
    • former_element + target_element { }
    • former요소 다음 위치에 해당하는 target 요소 선택
  11. General sibling combinator
    • former_element ~ target_element { }
    • 전후 요소의 부모가 같고 후자 요소가 뒤에 있을 때 선택
